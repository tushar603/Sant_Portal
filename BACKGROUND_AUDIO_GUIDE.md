# Background Audio for Home Page

## File to Add:
Place a background audio file in `public/audio/` folder with one of these names:
- **`background-music.mp3`** (recommended)
- **`background-music.m4a`**
- **`background-music.webm`**

## Supported Formats:
✅ **MP3** - Most compatible, works everywhere
✅ **M4A** - Good quality, Apple devices
✅ **WebM** - Modern browsers, good compression

## Browser will automatically choose the best format available!

## Suggested Audio:
- Peaceful devotional music
- Instrumental bhajan
- Om chanting
- Flute music
- Temple bells sound

## Where to Get:
1. YouTube - Search "devotional background music instrumental"
2. FreeSounds.org - Royalty-free spiritual music
3. Record temple ambience
4. Use any calm spiritual instrumental

## Audio Requirements:
- Format: MP3 or M4A
- Duration: 2-5 minutes (will loop)
- Volume: Moderate/Low
- Style: Calm, peaceful, non-intrusive
- Size: Under 5MB

## File Name:
Save as: `background-music.mp3` in `public/audio/` folder
