# Google Maps Integration Guide

## How to Add Google Maps to Saint Profiles

### Step 1: Go to Google Maps
Visit: https://www.google.com/maps

### Step 2: Search for Location
Search for the saint's birthplace (e.g., "Dehu, Pune" for Sant Tukaram)

### Step 3: Get Embed Code
1. Click the **Share** button
2. Click **Embed a map** tab
3. Copy the **iframe src URL** (the part inside quotes after src=)

### Example:
```
https://www.google.com/maps/embed?pb=!1m18!1m12!1m3!1d3782.123...
```

### Step 4: Add to Admin Panel
1. Login to admin panel
2. When adding/editing a saint
3. Paste the URL in "Google Maps Embed URL" field

## Example Locations for Maharashtra Saints:

### Sant Tukaram - Dehu, Pune
Search: "Dehu, Pune, Maharashtra"

### Sant Dnyaneshwar - Alandi, Pune
Search: "Alandi, Pune, Maharashtra"

### Sant Eknath - Paithan, Aurangabad
Search: "Paithan, Aurangabad, Maharashtra"

### Sant Namdev - Narsi Bamani, Hingoli
Search: "Narsi Bamani, Hingoli, Maharashtra"

### Sant Ramdas - Jamb, Jalna
Search: "Jamb, Jalna, Maharashtra"

## Tips:
- Use the exact birthplace name for accurate results
- The map will show automatically on the saint's detail page
- Map is responsive and works on mobile devices
- Users can interact with the map (zoom, pan, etc.)

## Note:
If you don't add a map URL, the map section simply won't appear on the saint's page.
